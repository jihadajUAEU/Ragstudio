<threat_model>
Assets: query evidence payloads, chunk/source text, metadata, raw JSON, and
operator trust in evidence provenance.
Threats: rendering untrusted metadata as HTML, implying missing data is healthy,
losing raw evidence fallback, duplicating viewer behavior by page, mobile drawer
overflow, or failing to restore focus to the source row button.
Mitigations: render all payload values as React text/JSON, use guarded
normalizers for loose records, keep raw JSON lower in the drawer, use shared
focus-trapped drawer shell, show explicit `Not recorded` states, and test Query
entry point selection.
Residual risk: Chunk Inspector integration and full graph missing-state polish
are completed in Plan 03.
</threat_model>
