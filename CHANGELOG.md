# Changelog

All notable changes to RAG-Anything Studio will be documented in this file.

## Unreleased

### Added

- Added the production runtime foundation for RAG-Anything: Postgres/PGVector defaults, Neo4j local runtime service, runtime profile fields, health diagnostics, mirrored chunk metadata, destructive runtime reindex seams, and runtime-backed query traces.

## [0.1.0.0] - 2026-05-08

### Added

- Added a standalone RAG-Anything Studio with document upload, automatic indexing, chunk inspection, scoped query runs, and source traces.
- Added variant creation, experiment execution, evaluation set import, scoring, comparison, and optimizer workflows so users can compare RAG strategies.
- Added pipeline, graph, diagnostics, and settings screens for understanding the full RAG flow and runtime configuration.
- Added FastAPI backend services, React frontend, static serving, API schema generation, and end-to-end smoke coverage.
- Added user and design documentation for the studio workflow, technology choices, and Superpowers planning artifacts.
